Tag Usage and Examples

The <table> element contains content that is used to represent a two-dimensional table consisting of rows and columns.

The <thead> element is used to define the headings of table columns encapsulated in table rows.

The <tbody> tag is a semantic tag that houses all table data other than table heading and table footer content.

The <tfoot> tag uses table rows to give footer content or to summarize content at the end of a table.

This <tr> element is used to add rows to a table before adding table data and table headings.

This element <th> is used to add titles to rows and columns of a table and just like the <td> element, it must be enclosed in a <tr> element.

This <td> element can be nested inside a <tr> element to add a cell of data to a table.

This colspan attribute on a <th> or <td> element indicates how many columns that particular cell should span within the table. Its value will take any positive integer between 1 and 1000 with 1 being the default value.

Just like the colspan attribute, the rowspan attribute on a <th> or <td> element indicates how many rows that particular cell should span within the table. While the default value is set to 1, it will take any positive integer up to 65534.

Recuerda

  • Always nest the <th> or <td> elements inside of a <tr> element before adding any form of information to the table
  • These tags, <thead>, <tbody>, and <tfoot> are important semantic tags used in organizing the structure of content within a table
  • Remove adjacent cells when applying the colspan or rowspan attributes to the <th> or <td> elements for a cleaner and more predictable layout
  • Nest additional <td> or <th> child elements within the same <tr> parent element to add cells horizontally (in a row) or in adjacent/different <tr> elements to add cells vertically (in a column)
